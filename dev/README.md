# Local Development

## Usage

1. Populate `GF_SECURITY_ADMIN_USER` and `GF_SECURITY_ADMIN_PASSWORD` in [.env](.env).

2. Start the containers using Docker compose:

   ```bash
   docker-compose up -d
   ```
